# vr driver

Драйвер для SteamVR. Эмулирует шлем виртуальной реальности на монитор, ориентирование в пространстве проиходит с помощью обычной мыши. Также эмулируется левый контроллер, в качестве его инпутов используется клавиатура, с которой драйвер может принимать 16 вводов. С помощью этого можно забиндить кнопки любой VR игры стима в SteamVR и игать в VR-режиме без какой-либо экипировки.
Весь код находится в папке src.
hmd_device_driver отвечает за эмуляцию шлема и вывод обычного (1920х1080) изображения на монитор.
controller_device эмулирует контроллер и принимает вводы с клавиатуры (только Windows).
device_provider запускает все наши девайсы, получает все вводы, рассчитывает движение мыши.
